# 🦉 Wit (Web Insight & Transcript)

[![Gemini CLI](https://img.shields.io/badge/Gemini_CLI-Skill-blue?style=flat-square)](https://geminicli.com)
[![Node.js](https://img.shields.io/badge/Node.js-v18+-green?style=flat-square)](https://nodejs.org)

**Wit**은 웹페이지, 공식 문서, 기술 블로그를 심층 분석하여 핵심 인사이트를 추출하고 전문적인 요약 및 번역본을 제공하는 Gemini CLI 전용 스킬입니다.

## 🚀 주요 기능

### 1. 심층 웹 분석 (`/wit:analyze`)
단순한 요약을 넘어 문서의 **도메인(분야)**을 파악하고 그에 맞는 인사이트를 제공합니다.
- **도메인 특화 요약**: IT, 금융, 뉴스 등 분야에 맞는 핵심 논지 분석
- **핵심 예제 추출**: 공식 문서의 주요 명령어(CLI) 및 코드 예제 우선 정리
- **구조적 브레이크다운**: 전체 요약(Executive Summary) 및 문단별 상세 내용 제공

### 2. 전문 필사 및 번역 (`/wit:translate`)
마크다운 구조를 유지하며 기술 용어의 맥락을 살린 고품질 번역을 수행합니다.
- **용어 보존**: 전문 용어 병기 및 코드 블록 원형 유지
- **자연스러운 의역**: 문맥을 반영한 매끄러운 번역 품질

## 🛠 설치 및 준비 사항

### 요구 사항
- **Gemini CLI** 인스턴스
- **Node.js v18** 이상 (내장 `fetch` API 사용)

### 설치 방법
본 저장소를 `.gemini/skills/wit` 경로로 복사하거나 심볼릭 링크를 연결합니다.
```bash
# 워크스페이스에서 스킬 등록
gemini skills link . --scope workspace
```

## 📖 사용 방법

### 명령어 호출
| 명령어 | 설명 | 예시 |
| :--- | :--- | :--- |
| `/wit:analyze` | 웹페이지 요약 및 분석 | `/wit:analyze <url> [output.md]` |
| `/wit:translate` | 웹페이지 전문 번역 | `/wit:translate <url> [ko\|en\|...] [translated.md]` |

### 자연어 자동 위임 (Auto-delegation)
커맨드를 입력하지 않아도 Gemini에게 자연어로 요청하면 적절한 에이전트가 자동 호출됩니다.
- *"이 링크(URL) 내용 요약하고 핵심 예제 좀 뽑아줘"*
- *"이 기술 블로그 한글로 매끄럽게 번역해줘"*

## 🏗 에이전트 아키텍처
Wit은 5개의 전문 서브 에이전트가 협업하여 최상의 결과물을 만듭니다.

1.  **Reader**: Node.js 스크립트로 고속 다운로드 및 본문 정제
2.  **Analyzer**: 문서 분야(Domain) 식별 및 핵심 인사이트 도출
3.  **Summary**: 구조적 요약 및 실전 예제/명령어 추출
4.  **Translator**: 문맥 유지 기술 문서 전문 번역
5.  **Formatter**: 마크다운 템플릿 기반 최종 보고서 생성
